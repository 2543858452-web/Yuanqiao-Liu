# Projected Riemannian Diffusion Sampling (PRDS)


本项目是论文《流形扩散模型的理论研究》的官方代码实现。

PRDS 是一种通用的流形采样框架，通过在切空间（Tangent Space）进行得分匹配，确保采样轨迹严格附着于流形表面。该框架基于 **Bakry-Émery 几何收缩性质** 建立了统一的收敛界，支持包括紧致流形（如球面）与非紧流形（如 SPD 矩阵流形）在内的多种几何空间。

---

## 核心特性

* **几何一致性**：利用指数映射与正交投影，规避流形外投影的高昂校正成本。
* **普适性**：统一支持紧致与非紧致黎曼流形。
* **理论完备**：证明了算法在 Wasserstein-1 距离下达到极小极大最优率（Minimax Optimal Rate）。
* **本征依赖**：收敛性能仅取决于流形本征维度 $d$，有效克服“维数灾难”。
